## NON-NEGOTIABLE RULES

### Rule #1

DO NOT CHANGE THE HOMEPAGE CATEGORIES.

The following categories are permanently locked:

- Madrid Essentials
- Living in Madrid
- See in Madrid
- Do in Madrid
- Madrid Now

These are constitution-level decisions.

### Rule #2

Madrid first. Spain second.

Examples:

GOOD:
- From Madrid to Toledo
- From Madrid to Segovia
- From Madrid to El Escorial

BAD:
- General Spain Tourism
- Generic Spain Travel Guides

### Rule #3

Build RECORDS, not pages.

Preferred:
- Metro de Madrid
- Prado Museum
- Atocha Station
- NIE/TIE
- Healthcare

Every record should contain:
- Name
- Category
- Status
- Official URL
- Related Records
- English Friendly flag

### Rule #4

Prioritize English-speaking users.

Examples:
- English-speaking doctors
- English-speaking lawyers
- English movie screenings
- English-friendly housing

### Rule #5

'Arrive, Survive, Live, Explore, Enjoy' are NOT homepage categories.

These are secondary:
- Tags
- Filters
- User journeys
- Internal navigation

They must never replace the five homepage categories.
