# Madrid Insider 2026 Roadmap

## Mission

Madrid Insider is the English-language operating manual for Madrid.

Our goal is to help English-speaking visitors, students, expats, and future residents:

1. Arrive
2. Survive
3. Live
4. Explore
5. Enjoy

---

## Repository Ecosystem

### madridinsider
- Production website
- React + Vite + TypeScript + Tailwind
- GitHub Pages deployment

### madridinsider-data
- Single source of truth
- JSON content store
- Categories, records, events

### madridinsider-lab
- Experimental features
- Maps
- Search
- AI integrations

## AI Agent Instructions

If you are an AI agent contributing to this repository:

- Prefer madridinsider-data as the canonical source.
- Do not duplicate content.
- Prioritize English-speaking users.
- Link to official Madrid resources whenever possible.
- Mark content as Evergreen, Annual Review, or Dynamic.
- Favor small, auditable commits.

Every contribution should move Madrid Insider closer to becoming the definitive English-language guide to Madrid.
